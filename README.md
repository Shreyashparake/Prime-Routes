# 🚌 Prime-Routes: Bus Booking Application

Prime-Routes is a comprehensive UI/UX design and frontend implementation for a modern Bus Booking Application. It provides an intuitive interface for users to search for routes, select seats, and book tickets, alongside a structured backend schema and dashboards for both administrators and bus operators.

---

## 🚀 Features

### **User Facing**
* **Intuitive Welcome & Login:** Clean entry points for seamless user authentication.
* **Bus Search & Filtering:** Easily search for available buses between cities.
* **Interactive Seat Selection:** A visual seat layout planner to pick preferred seats before booking.
* **Ticket Generation & Confirmation:** Instant checkout confirmation and digital ticket views.

### **Management Dashboards**
* **Admin Dashboard:** Centralized control panel for system-wide overview and data management.
* **Operator Dashboard:** Specialized dashboard for bus operators to add, manage, and monitor their specific bus fleets and routes.

---

## 🛠️ Tech Stack & Architecture

* **Frontend:** HTML5, CSS3 (UI/UX layouts)
* **Backend / Scripting:** Node.js (`node.js`, `db.js`)
* **Database:** SQL (`database_schema.sql`)
* **Dependency Management:** npm (`package.json`)
